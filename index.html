http4s-jdk-http-client - JDK 11+ http client implementation for http4s clients

Quick Start

To use http4s-jdk-http-client in an existing SBT project with Scala 2.11 or a later version, add the following dependencies to your build.sbt depending on your needs:

libraryDependencies ++= Seq(
  "org.http4s" %% "http4s-jdk-http-client" % "<version>"
)
